# Code of Conduct

## Our Pledge

We want `opencode-sandbox` to be a respectful, technically serious, and inclusive project.
Contributors should help create an environment where people can participate without harassment.

## Our Standards

Examples of behavior that contribute to a positive environment:

- giving constructive technical feedback
- assuming good faith while still being precise
- accepting responsibility and correcting mistakes
- respecting different experience levels and backgrounds

Examples of unacceptable behavior:

- harassment, insults, or personal attacks
- deliberate intimidation
- publishing private information without permission
- bad-faith disruption of discussions or reviews

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Reporting

Please report conduct issues privately to the project maintainers.

## Scope

This Code of Conduct applies within project spaces and in public spaces when someone is representing the project.
