# Security Policy

## Reporting a vulnerability

Please do not open a public issue for security-sensitive reports.

Instead, share:

- a clear description of the issue
- the affected version or commit
- reproduction steps
- impact assessment if known

Until a dedicated private reporting channel exists, contact the maintainers directly and keep the report non-public.

## Supported versions

This project is currently maintained on a rolling basis. The latest commit on the default branch is the supported version.
